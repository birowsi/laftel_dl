사용 순서

처음 쓰더라도 흐름은 단순합니다. 아래 순서만 따르면 됩니다.

  1. 세션 확보를 눌러 로그인 상태를 확인합니다. 필요하면 브라우저 창이 열리고, 로그인과 프로필 선택을 직접 완료하면 됩니다.
  2. 작품 URL의 숫자 부분을 작품 ID에 넣습니다. 전체 다운로드면 회차 범위는 비워두고, 일부만 받을 때만 1-3,5,7 형태로 적습니다.
  3. 다운로드 시작 후에는 실시간 로그 섹션을 보면 됩니다. 완료 뒤엔 분할압축 섹션에서 바로 500MB 아카이브를 만들 수 있습니다.
